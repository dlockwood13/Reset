# Reset

A privacy-first PWA playbook for navigating job search after redundancy in the UK.

## Features
- Installable on iOS/Android home screen
- Works fully offline after first load
- All data stored locally on device (nothing leaves your phone)
- Drill-down journey navigation
- Daily checklist with streak tracking
- Mood-aware journal
- Export your data anytime as JSON
